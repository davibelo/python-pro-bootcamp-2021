to work with JSON files use:

import json

to write:
    json.dump()
to read:
    json.load()
to update
    json.update()

those methods can be used inside a "open" command call:

see 13-passwordManager project:

try:
    with open(DATA_FILE, mode="r") as data_file:
        # reading old data
        data = json.load(data_file)
except FileNotFoundError:
    with open(DATA_FILE, mode="w") as data_file:
        # creating a data file and saving the new_data
        json.dump(new_data, data_file, indent=4)
else:
    # updating data with new data
    data.update(new_data)
    with open(DATA_FILE, mode="w") as data_file:
        # saving updated data
        json.dump(data, data_file, indent=4)
